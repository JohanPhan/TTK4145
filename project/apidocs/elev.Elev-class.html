run(self)

source code 

Start communication with master, movement- and button handler.

Elev objects are used as a mirror of the real clients on the master side, so to initiate connection with the server, move the elevator, etc this must be called.

close(self)

source code 

Cleanup function.

Should always be called explicitly when an elevator object goes out of scope.

insert_task(self, floor)

source code 

Insert a given floor at the correct place in the queue.

Considers the current state and direction of the elevator and places the new floor at the "best" position in the queue.

Args: floor: The floor to be inserted

next_dir(self)

source code 

Calculates the next direction of the elevator.

Returns: Either DIRN_UP, DIRN_DOWN or DIRN_STOP. All defined in constants.py.

movement_handler(self)

source code 

Monitor the queue and handle movement to the next task.

Runs in its own thread, continuously monitoring the queue and moves the elevator in the direction of the next task.

button_handler(self)

source code 

Monitor buttons and handle the pressing of said buttons.

Runs in its own thread, continuously checks the systems buttons, and if a button is pressed it handles it accordingly.
